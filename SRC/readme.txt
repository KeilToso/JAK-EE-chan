The Arduino IDE must be used for writing code and compiling. Version 1.6.7 is the latest compatible version at this time. 
Teensyduino 1.28 must be installed to interface with the Teensy. Ensure that the Teensy device is selected under "Tools -> Board: "
Installing all libraries when installing Teensyduino will provide the required FastLED library as well as the FFT functions.
